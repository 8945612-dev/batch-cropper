# JavaFX Batch Cropper

Современное desktop-приложение на JavaFX для пакетной нарезки изображений по интерактивной сетке.

## Что умеет

- загружать сразу много изображений
- показывать превью выбранного файла
- настраивать сетку мышкой прямо на превью
- тянуть внешние границы области crop
- тянуть внутренние вертикальные и горизонтальные линии
- менять число строк и колонок
- применять одну и ту же разметку ко всей пачке файлов
- сохранять каждую ячейку как отдельное изображение
- экспортировать в PNG или JPG

## Как работает сетка

Сетка хранится в относительных координатах, а не в пикселях.
Это значит, что одна и та же разметка применяется к изображениям разного размера, если у них одинаковая композиция.

## Требования

- JDK 21
- Maven 3.9+

## Запуск

В корне проекта:

```bash
mvn clean javafx:run
```

## Сборка jar

```bash
mvn clean package
```

## Что внутри проекта

- `BatchCropperApp.java` — основное окно и UI
- `PreviewPane.java` — превью и drag-and-drop логика для сетки
- `GridModel.java` — модель сетки в относительных координатах
- `BatchCropExporter.java` — пакетный экспорт обрезков
- `app.css` — стили интерфейса

## Ограничения текущей версии

- используется равномерная исходная разметка при смене количества строк/колонок
- поддерживаются форматы чтения/записи, доступные через стандартный Java ImageIO
- EXIF-ориентация и форматы вроде WebP специально не добавлялись, чтобы проект остался без лишних зависимостей
